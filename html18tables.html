HTML-Table

Table cell defined by td stands for table data

A table cell contain all sorts of HTML elements: text, images,lists,links,etc.

tr stands for table row

th stands for table header, by default text is bold and centered

round table borders , use border-radius:20px

for table borders use border-style property , values are dotted, dashed,solid,double,groove,ridge,inset,outset,none,hidden

for table column width use width at data cell using width property

for table row height use height at data cell using height property

caption inserted after table

HTML tables adjust padding inside cells, and also space between the cells

cell padding is space between cell edges and cell content by def it is set to 0 use padding property to set value

here we have padding-top, padding-bottom, padding-left, padding-right

Cell spacing is the space between each cell here here we use border-spacing property by Def it is 2 pixels

for Zebra Stripes effect one other we use tr:nth-child(even/odd) selector this is for horizontal

for Vertical Zebra Stripes, we use Like td:nth-child(even), th:nth-child(even)

for both hor and ver zebra stripes use both using transparent ie. rgba

if we have only border at bottom use border-bottom

:hover selector on tr to highlight table rows on mouse over

use tag that spans 2 columns

Hide columns by using style="visibility:collapse"
