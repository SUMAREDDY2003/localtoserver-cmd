BLOCK AND INLINE

Every HTML element has default display value : block & inline

BLOCK-LEVEL Elements

Always start on a new line, and browsers automatically add some space before and after the element

Ex: div,p,address, aside ,canvas,div,dt,figcaption,footer,h1-h6,main,noscript,section,tfoot,video,article,blockquote,dd,dl,fieldset,figure,form,header,li,nav,ol,pre,table,ul

INLINE-LEVEL Elements

Doesn't start on a newline only takes up as much as width necessary

Ex: a, acronym,bdo,br,cite,dfn,i,input,label,object,q,script,small,strong,sup,time,var,abbr,b,big,button,code,em,img,kbd,map,output,samp,select,span,sub,textarea, tt
